# Leetcode 101 Problem solving Notes



This book is a problem solving notes for **leetcode101**, a book published by Chang Gao. In this book

The aim of creating this book is to summarize skils of solving problem and records how to analyze the problem and covert problem solving idea to code.

In this book, a question will belongs to a pages and a pages will contents the following sections:

* Senction A: question link and question sequence.
* Section B: analize of this problem and problem solving idea
* Section C: how to abstract idea to code.
* Section D: code
* Section E(optional): tips
